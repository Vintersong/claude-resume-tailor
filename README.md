# claude-resume-tailor
What it says on the tin.

resume-tailor

A small resume-tailoring skill and template repo for generating ATS-friendly resumes from one master reference file.
What it includes

    SKILL.md — instructions for analyzing a job post and tailoring a resume.

    references/master_resume.md — a public template users can copy and fill in with their own details.

    references/pdf_generation.md — notes for the PDF layout and JSON schema.

    scripts/generate_resume_pdf.py — a simple ReportLab-based PDF generator.

How it works

    Add your real experience to references/master_resume.md.

    Provide a job posting.

    Tailor the resume to the role.

    Export the result as an ATS-friendly PDF.

Notes

    The public template is sanitized and contains no personal information.

    Replace all placeholders before using the template for a real application.

    This repo is meant to be copied and adapted to your own workflow.
